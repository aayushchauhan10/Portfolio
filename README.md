
# Ayush Portfolio
A pixel-perfect, classic portfolio built with Next.js (App Router), TailwindCSS, Lenis (smooth scroll), GSAP, Framer Motion, and React Spring. Dark/light mode via next-themes (default: dark). Includes starry background + aurora.

## Quickstart
```bash
pnpm i   # or npm i / yarn
pnpm dev # or npm run dev / yarn dev
```
Open http://localhost:3000

## Customize
Edit `data/resume.js` to update content pulled from your resume.
